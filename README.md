# Air Quality Index (AQI) Dashboard

## Project Overview
Built an interactive Power BI dashboard using a Kaggle Air Quality Index (AQI) dataset to analyze pollution levels, pollutant concentration, and city-wise air quality trends across different regions.

The project focuses on identifying major pollutants, comparing AQI levels across cities, and highlighting environmental risk patterns through interactive visualizations and KPI metrics.


## Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization


## Key Features
- City-wise AQI Analysis
- PM2.5 & PM10 Pollution Tracking
- Interactive KPI Cards and Filters
- Pollution Trend Visualization
- Comparative AQI Analysis Across Cities


## Data Analysis Process
- Cleaned and transformed AQI dataset using Power Query
- Created DAX measures for pollution metrics
- Built interactive dashboard visualizations
- Analyzed pollutant concentration and city-level trends


## Key Insights
- PM2.5 and PM10 were identified as dominant pollutants
- Pune, Mumbai, and Sangli showed higher AQI levels compared to less populated cities
- Highlighted environmental risk trends in urban regions
- Recommended stricter industrial controls, green transportation, and expanded air quality monitoring
